Админка привилегий
------------------

- Страницы с регулировкой конкретных прав:
  - Страница привилегий конкретного пользователя в конкретном ивенте (EUPriv)
    - Выводить только переопределенные привилегии, непереопределенные выводить неактивным образом
    - Контрол для переопределения привилегии
    - Контрол для сброса в дефолтное состояние
  - Страница глобальных привилегий конкретного пользователя (GUPriv)
    - Выводить только переопределенные привилегии по умолчанию, непереопределенные выводить неактивным образом
    - Контрол для переопределения привилегии
    - Контрол для сброса в дефолтное состояние
  - Страница привилегий конкретной группы в конкретном ивенте (EGPriv)
    - Выводить только переопределенные привилегии, непереопределенные выводить неактивным образом
    - Контрол для переопределения привилегии
    - Контрол для сброса в дефолтное состояние
  - Страница глобальных привилегий конкретной группы (GGPriv)
    - Выводить только переопределенные привилегии по умолчанию, непереопределенные выводить неактивным образом
    - Контрол для переопределения привилегии
    - Контрол для сброса в дефолтное состояние



- Страницы со списками:
  - Страница со списком пользователей, имеющих переопределенные привилегии для конкретного ивента (EUList)
    - В списке ссылки на EUPriv
    - Ссылка на страницу добавления нового сета переопределенных привилегий EUPriv
  - Страница со списком групп, имеющих переопределенные привилегии для конкретного ивента (EGList)
    - В списке ссылки на EGPriv
    - Ссылка на страницу добавления нового сета переопределенных привилегий EGPriv
  - Страница со списком ивентов, имеющих переопределенные привилегии для конкретного пользователя (UEList)
    - В списке ссылки на EUPriv
    - Ссылка на страницу добавления нового сета переопределенных привилегий EUPriv
  - Страница со списком ивентов, имеющих переопределенные привилегии для конкретной группы (GEList)
    - В списке ссылки на EGPriv
    - Ссылка на страницу добавления нового сета переопределенных привилегий EGPriv
  - Страница со списком пользователей, имеющих переопределенные глобальные привилегии (GUList)
    - В списке ссылки на GUPriv
    - Ссылка на страницу добавления нового сета переопределенных привилегий GUPriv
  - Страница со списком групп, имеющих переопределенные глобальные привилегии (GGList)
    - В списке ссылки на GGPriv
    - Ссылка на страницу добавления нового сета переопределенных привилегий GGPriv
- Поисковая страница (Search) - точка входа:
  - Поиск ивентов
    - В результатах ссылки на EUList, EGList
  - Поиск юзеров
    - В результатах ссылки на UEList + GUList
  - Поиск групп (здесь может быть и тупо список)
    - В результатах ссылки на GEList + GGList
  - Ссылки напрямую на GUList, GGList
